Technologies we work with

Python: We have lots of experience in Python programming doing developments from simple scripts to full complex applications.

Qt: We work in several Qt projects related with KDE and we have Qt certified developers. We have experience with Qt and KDE tools and C++ development.

Gtk+: We work in several GTK+ projects related with GNOME. Some of us are GNOME developers and are experts in GTK+ and GNOME tools, with C and Python languages.

Django: As python developers we use the Django web framework to develop web applications and services. netfilter

Netfilter: Netfilter is a set of hooks inside the Linux kernel that allows kernel modules to register callback functions with the network stack. A registered callback function is then called back for every packet that traverses the respective hook within the network stack.

SDL: We develop several videogames using SDL and C++.

Linux: As free software developers we center our development in a Linux environment and we work with a lot of technologies related to it. We work with bash scripting, linux distribution packaging, linux desktop development and other GNU/Linux related technologies.
